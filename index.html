⚡ Supabase 연결 설정

실시간 채팅과 멀티 참여를 위해 Supabase 프로젝트가 필요해요.
아래 3단계만 따라하면 5분 안에 완료돼요! (완전 무료)

1
Supabase 프로젝트 만들기

→ supabase.com 접속 → GitHub 로그인 → "New project" → 이름·비밀번호·리전(Seoul) 입력 → 생성

2
SQL 한 번 실행해서 테이블 만들기

→ 왼쪽 메뉴 "SQL Editor" → 아래 코드 그대로 복붙 → Run

create table rooms (code text primary key, data jsonb not null default '{}'::jsonb, updated_at timestamptz default now()); alter table rooms enable row level security; create policy "all" on rooms for all using (true) with check (true); alter publication supabase_realtime add table rooms;
3
API 키 복사

→ ⚙️ Settings → API → Project URL과 anon public 키를 아래에 붙여넣기

설정값은 이 기기에만 저장돼요. 한 번 설정하면 다음부턴 자동으로 연결됩니다.
